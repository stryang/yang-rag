# 开发最佳实践指南

## 1. Git 工作流

### 1.1 分支命名规范
```
feature/<ticket>-brief-description
bugfix/<ticket>-brief-description
hotfix/<ticket>-brief-description
release/v1.0.0
```

### 1.2 Commit 规范
```
<type>(<scope>): <subject>

<body>

<footer>
```

**Type 类型:**
- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档更新
- `style`: 代码格式
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具

**示例:**
```bash
feat(user): add user profile API

- GET /users/:id
- PATCH /users/:id
- response with avatar_url

Closes #123
```

### 1.3 Pull Request 流程
1. 从 `develop` 创建功能分支
2. 开发并提交代码
3. 推送分支并创建 PR
4. 至少 2 人代码审查通过
5. 合并到 `develop`
6. 部署到预发布环境
7. 测试通过后合并到 `main`

## 2. 数据库设计

### 2.1 表命名
```sql
-- 统一使用单数名词
CREATE TABLE user (
    id BIGSERIAL PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    email VARCHAR(255) NOT NULL UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引命名
CREATE INDEX idx_user_email ON user(email);
CREATE INDEX idx_user_status_created ON user(status, created_at);
```

### 2.2 迁移脚本
```python
# migrations/001_create_user_table.py

def upgrade(db):
    db.execute("""
        CREATE TABLE user (
            id BIGSERIAL PRIMARY KEY,
            username VARCHAR(50) NOT NULL UNIQUE,
            email VARCHAR(255) NOT NULL UNIQUE,
            password_hash VARCHAR(255) NOT NULL,
            status VARCHAR(20) DEFAULT 'active',
            created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
            updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
        )
    """)
    db.execute("CREATE INDEX idx_user_email ON user(email)")

def downgrade(db):
    db.execute("DROP TABLE IF EXISTS user")
```

## 4. 安全实践

### 4.1 密码处理
```python
import bcrypt

def hash_password(password: str) -> str:
    """密码哈希"""
    salt = bcrypt.gensalt()
    return bcrypt.hashpw(password.encode(), salt).decode()

def verify_password(password: str, hashed: str) -> bool:
    """密码验证"""
    return bcrypt.checkpw(password.encode(), hashed.encode())
```

### 4.2 SQL 注入防护
```python
# Good - 使用参数化查询
cursor.execute(
    "SELECT * FROM users WHERE email = %s",
    (email,)
)

# Bad - 字符串拼接
cursor.execute(
    f"SELECT * FROM users WHERE email = '{email}'"
)
```

### 4.3 XSS 防护
```python
from markupsafe import escape

# 在模板中
{{ user_input | escape }}

# API 响应
return JSONResponse({
    "content": escape(user_content)
})

from functools import lru_cache
import redis

@lru_cache(maxsize=1000)
def get_product_category(product_id: int) -> str:
    """产品分类（带缓存）"""
    return db.query(
        "SELECT category FROM products WHERE id = %s",
        (product_id,)
    )

# Redis 缓存
def get_user_with_cache(user_id: int) -> User:
    cache_key = f"user:{user_id}"
    cached = redis.get(cache_key)

    if cached:
        return User.parse_raw(cached)

    user = db.get_user(user_id)
    redis.setex(cache_key, 3600, user.json())

    return user
```
